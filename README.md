### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`layui`、`HTML` / `CSS` / `Javascript`/ `bootstrap` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：系统管理员、招录员、学生。

**学生功能：**
1. 登录
2. 注册
3. 个人信息管理
4. 报名资料填写
5. 查看考试信息
6. 查看准考证
7. 成绩查询


**招录员功能：**
1. 登录
3. 个人信息管理
4. 报名管理
5. 成绩管理
6. 准考证管理



**管理员功能：**
1. 登录
2. 个人信息管理与查看
3. 报名管理
4. 成绩管理
5. 考生管理
6. 准考证管理
7. 管理员管理
